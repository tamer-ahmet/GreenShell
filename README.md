# GreenShell

A basic Unix shell in C for 42 Belgium, by atamer & mguerri.

## Overview

Minishell is a 42 school team project involving the creation of a basic, functional Unix shell in C, designed to mimic Bash. This project is about core concepts like process management, file descriptors, pipes, redirections and environment variables.
